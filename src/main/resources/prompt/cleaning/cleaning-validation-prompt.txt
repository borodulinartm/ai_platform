CORE VALIDATION PRINCIPLES
==================================================

Validation MUST be conservative.

NEVER guess truncation semantically.
NEVER reject text because it "looks noisy".
NEVER reject text because it "feels incomplete".

Meaningful article text is ALWAYS allowed unless explicitly forbidden.

==================================================
STRICT VIOLATIONS
==================================================

--------------------------------------------------
V001_MARKERS_PRESENT
--------------------------------------------------

FAIL if output contains:
- ----BEGIN TEXT-------
- ----END TEXT---------

--------------------------------------------------
V002_FORBIDDEN_HTML
--------------------------------------------------

FAIL if output contains ANY HTML tag except <img>.

Examples:
- <div>
- <p>
- <span>
- <script>
- <style>
- <form>
- <button>
- <html>
- <body>

IMPORTANT:
Escaped HTML text is allowed.

--------------------------------------------------
V003_INVALID_IMG_TAG
--------------------------------------------------

FAIL if an <img> tag:
- is syntactically malformed
- contains nested tags
- is structurally incomplete

VALID:
<img src="a.jpg">
<img src="a.jpg"/>

INVALID:
<img
<img src="
<img><div>

IMPORTANT:
Do NOT validate network accessibility.
Do NOT invent broken-image failures.

--------------------------------------------------
V004_RAW_URL_PRESENT
--------------------------------------------------

FAIL if standalone raw URLs exist outside <img> tags.

Examples:
https://example.com
http://abc.org
www.test.com

IMPORTANT:
URLs inside img src are allowed.

--------------------------------------------------
V005_MARKDOWN_PRESENT
--------------------------------------------------

FAIL if Markdown formatting syntax exists.

Examples:

**bold**

```code```
[link](url)

IMPORTANT:
- Single "*" inside text is allowed
- Hyphen lists are allowed
- Mathematical notation is allowed

- "Analysis:"
- "Reasoning:"
- "The result is"

IMPORTANT:
Only explicit meta assistant text counts.

--------------------------------------------------
V007_EMPTY_OUTPUT

"NO_CONTENT" is VALID.

--------------------------------------------------
